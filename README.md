# fenwick

дерево Фенвика с применением для обработки поддеревьев в графах

SimpleList - реализация списка без использования коллекций Java
GraphWithFenwick - граф с поддержкой операций через дерево Фенвика: построение графа (ориентированного/неориентированного); обход в глубину; операции с поддеревьями через дерево Фенвика
